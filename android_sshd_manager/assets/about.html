DropBearServer II
by shk.schneider

DropBearServer II is a minimalist yet powerfull SSH server for Android.
It uses the dropbear's sources, compiled for ARM architecture (https://matt.ucc.asn.au/dropbear/dropbear.html).

DropBearServer II supports ssh with root permissions, password, public keys but mainly: ssh from and to phone and scp from and to phone.

DropBearServer II is Free and Open-Source.
